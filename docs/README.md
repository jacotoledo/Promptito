# Landing Page

This folder contains the GitHub Pages landing page for Promptito.

**Live at:** https://jacotoledo.github.io/Promptito

For custom domain, add your domain to `CNAME`.

---

To update the landing page:
1. Edit `index.html`
2. Push to GitHub
3. GitHub Pages will auto-deploy

Settings → Pages → Source: Deploy from a branch → main / (root) → /docs
