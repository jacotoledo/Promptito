Why Promptito?

No More Chaos

Stop naming files "v2_FINAL_real.md". Simple folders, one search bar, done.

Zero Dependencies

It's a single Go binary. No Node.js, no Python, no database. Just run it.

Privacy First

No tracking, no analytics, no telemetry. Your prompts stay on your server.

Standards Compliant

Built on real AI metadata standards: IPTC, ISO/IEC 5259, NIST AI RMF, SFIA 9.

AI Agent Ready

REST API with clean endpoints. Your AI agents can fetch prompts programmatically.

Free Forever

Deploy on Koyeb's free tier. No credit card. No server costs.

Quick Start

Option 1: Free Website (Recommended)

  1. Fork this repository on GitHub
  2. Go to app.koyeb.com and sign up
  3. Click Create Service → Deploy from GitHub
  4. Select your fork and click Deploy
  5. Done! Your prompts are live at https://your-app.koyeb.app

Option 2: Run Locally

  1. Download from GitHub Releases
  2. Double-click promptito.exe
  3. Open http://localhost

Option 3: Docker

  1. Run: docker run -p 80:80 ghcr.io/jacotoledo/promptito:latest
  2. Open http://localhost

API Reference

GET /api/skills List all prompts
GET /api/skills/{name} Get one prompt
GET /api/search?q=text Search prompts
GET /api/tags List all tags
GET /health Health check
